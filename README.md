# Simple file upload

Simple file upload sample code with JavaScript, HTML, CSS, Bootstrap 5 and PHP backend

## Step to run

* Download all files from this repository
* Keep the files inside a web server
* Create a folder named "uploads" inside the folder where downloaded file resides
* Start the web server
* Access `http://localhost:<configured_port>` to see the homepage for file upload

### Note: Files get uploaded to `uploads/` folder
